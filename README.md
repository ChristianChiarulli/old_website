# chris@machine website

## Usage

### Install Dependencies
```bash
npm install
```

### Run Dev Server (http://localhost:3000) 
```bash
npm run dev
```

### Caching

Husky is used to run a cache script on git commit. Caching is used for the search api route/serverless function
